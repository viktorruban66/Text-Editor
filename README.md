# 📝 Text Editor – консольный построчный редактор 

> Полноценный интерактивный текстовый редактор с поддержкой поиска, замены и базовой подсветки синтаксиса. Реализован на 7 языках программирования для демонстрации возможностей и сравнения.

## ✨ Возможности

- 📂 **Открытие/сохранение файлов** – передайте имя файла аргументом командной строки
- ✏️ **Построчное редактирование** – изменяйте, вставляйте, удаляйте строки
- 🔍 **Поиск по тексту** (с поддержкой regex в некоторых реализациях)
- 🔄 **Замена** – глобальная или построчная
- 🎨 **Подсветка синтаксиса** (ANSI) для языков C, C++, Java, Python, Go, Rust, JavaScript
- 💾 **Автоопределение изменений** – предупреждение при выходе без сохранения
- 📜 **История команд** – навигация стрелками вверх/вниз
- 🖥 **Полностью консольный интерфейс** – работает везде, где есть терминал

## 🚀 Быстрый старт

Выберите язык и запустите:

| Язык | Команда |
|------|---------|
| Python | `python editor.py [файл]` |
| Rust | `cargo run -- [файл]` |
| Go | `go run editor.go [файл]` |
| C++ | `g++ editor.cpp -std=c++17 -o editor && ./editor [файл]` |
| TypeScript | `npm install && npx ts-node editor.ts [файл]` |
| Java | `javac Editor.java && java Editor [файл]` |
| C# | `dotnet run [файл]` |

## 🎮 Команды редактора (все версии)

После запуска отображается текст с номерами строк. Ввод команд осуществляется через `:` или просто вводом номера строки.
